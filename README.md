# Plain Programming Language

Plain is a natural language programming language that transpiles to Python. Write code in plain English!

## Quick Start

### Install

**Easy way (recommended):**
```powershell
.\install_user.ps1
```

**Manual way:**
```powershell
pip install --user -e .
```

This installs to your user directory (no admin required). Restart PowerShell/VS Code after installation.

### Use

```powershell
plain run examples/hello.pln
```

## Features

- Natural English syntax
- Full Python support
- Auto-imports 50+ enterprise libraries
- Interactive REPL
- Compile to Python
- **Enterprise-ready**: Async/await, error handling, classes, decorators
- **Production-grade**: Suitable for Facebook, Instagram, TikTok, Amazon, Discord
- **Advanced features**: Generators, context managers, type hints, database operations
